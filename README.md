# PCL Intelligence Homepage

一个可以在 PCL2 启动器上使用 Google Gemini 或其他大模型的主页

## 使用方法

1. Fork 本仓库
2. 使用 Vercel 部署
    - 需要填写 API 密钥
    - Env Name: `api_key`
    - Env Value: `你的 API 密钥`
3. （可选）在 Vercel 绑定自定义域名
4. 在 PCL2 启动器选择 `联网主页`，输入 Vercel 部署的根 URL，例如 `https://abc.vercel.app` 或 `https://intelligence.pcl-community.org`
